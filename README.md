# Repository for the Fluid dynamics class at Utrecht University

This is a repository contains code for weekly exercises and hand-in projects that I worked on as a part of a course on Fluid Dynamics at UU. It also contains Tex code and the asociated pdf files for the projects.

## Structure

The code files are sorted into either the weekly exercises or project folders in the python directory.
The LaTeX files are sorted in the same way in the LaTeX directory.
