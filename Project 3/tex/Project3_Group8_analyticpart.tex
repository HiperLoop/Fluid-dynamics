\section{Analytic part}

\subsection*{a)}
To argue that we can use the streamfunction $\psi$ to describe the flow, we can start at the first boundary condition, namely that the fluid is incompressible. This implies that the density does not change with time. If we plug this into the continuity equation, we get:
\begin{align*}
    \frac{\partial \rho}{\partial t} + \nabla \cdot (\rho \mathbf{v}) = 0 \\
    \Leftrightarrow \nabla \cdot \mathbf{v} = 0
\end{align*}
This means that the velocity field is divergence free. We can now use this to define a streamfunction $\psi$ that automatically satisfies
\begin{align*}
    u &= \frac{\partial \psi}{\partial z}  \\
    w &= -\frac{\partial \psi}{\partial x}
\end{align*}
If we now insert these definitions into the boundary condition, we find:
\begin{align*}
    \partial _x \left( \frac{\partial \psi}{\partial z} \right) + \partial _z \left( -\frac{\partial \psi}{\partial x} \right) &= 0 \\
    \frac{\partial^2 \psi}{\partial x \partial z} - \frac{\partial^2 \psi}{\partial z \partial x} &= 0 \quad \square
\end{align*}
Since the streamfunction is function describing the position of the particle, its derivative is the velocity of the particle, so it also makes sense intuitively. 
We now plug this into the second and third equations:
\begin{align*}
    \textbf{Pr}^{-1}(\partial _t u+ u\partial_xu+
    w\partial _zu)=-\partial _xp+\partial ^2_xu+\partial ^2_zu\\
    \Rightarrow \textbf{Pr}^{-1}(\partial _t \partial_z \psi+ u \partial_x\partial_z \psi+w \partial _z^2\psi)=-\partial _xp+\partial ^2_x\partial_z\psi+\partial ^3_z\psi\\
    \textbf{Pr}^{-1}(\partial _t w+ u\partial _xw+w\partial _zw)=-\partial _zp+\partial ^2_xw+\partial ^2_zw+\textbf{Ra}\vartheta\\
    \Rightarrow \textbf{Pr}^{-1}(-\partial _t \partial_x \psi-u \partial_x^2 \psi-w \partial_z \partial_x \psi)=-\partial _zp-\partial ^3_x\psi-\partial ^2_z\partial _x\psi+\textbf{Ra}\vartheta
\end{align*}
To cancel out the pressure, we are now deriving both equations, the first one with respect to $z$ and the second one with respect to $x$:
\begin{align*}
    \textbf{Pr}^{-1}(\partial _t \partial^2_z \psi+ u \partial_x\partial^2_z \psi+w \partial^3_z \psi)=-\partial _{x}\partial_zp+\partial ^2_x\partial_z^2\psi+\partial ^4_z\psi\\
    \textbf{Pr}^{-1}(-\partial _t \partial^2_x \psi-u \partial^3_x \psi-w \partial_z \partial^2_x \psi)=-\partial _{x}\partial_zp-\partial ^4_x\psi-\partial ^2_z\partial _x^2\psi+\textbf{Ra}\partial _x\vartheta
\end{align*}
Now we can subtract the second equation from the first one to get:
\begin{align*}
    \textbf{Pr}^{-1}(\partial _t \partial^2_z \psi+ u \partial_x\partial^2_z \psi+w \partial^3_z \psi+\partial _t \partial^2_x \psi+u \partial^3_x \psi+w \partial_z \partial^2_x \psi)\\
    =\partial ^2_x\partial_z^2\psi+\partial ^4_z\psi+\partial ^4_x\psi+\partial ^2_z\partial _x^2\psi-\textbf{Ra}\partial _x\vartheta\\
    \Leftrightarrow \textbf{Pr}^{-1}(\partial _t (\partial^2_x+ \partial^2_z) \psi+ u \partial_x(\partial_x^2+\partial_z^2) \psi+w \partial_z (\partial_x^2+\partial_z^2) \psi)\\
    =\partial _x^2(\partial_z^2+\partial_x^2)\psi+\partial _z^2(\partial_x^2+\partial_z^2)\psi-\textbf{Ra}\partial _x\vartheta\\
\end{align*}
Now we can use the definition of the Laplacian, $\nabla ^2 = \partial_x^2 + \partial_z^2$, to get:
\begin{align*}
    \textbf{Pr}^{-1}(\partial _t \nabla^2 \psi+ u \partial_x \nabla^2 \psi+w \partial_z \nabla^2 \psi)=\partial_x^2(\nabla^2\psi)+\partial_z^2(\nabla^2\psi)-\textbf{Ra}\partial _x\vartheta \quad \square 
\end{align*}

\subsection*{c)}
We know that the solution is an equilibrium solution if all time-derivatives are zero. Therefore, we have to check whether 
\begin{align}
    \dot X=s(Y-X)=0 \label{goveqX}\\
    \dot Y=rX-Y-XZ=0 \label{goveqY}\\
    \dot Z=XY-bZ=0 \label{goveqZ}
\end{align}
given our initial $X_0,Y_0,Z_0$. We therefore check the three given solution. For $X_0=Y_0=Z_0=0$, we have:
\begin{align*}
    \dot X=s(0-0)=0\\
    \dot Y=r\cdot 0-0-0\cdot 0=0\\
    \dot Z=0\cdot 0-b \cdot 0=0 \quad \square
\end{align*}
Now we check the second solution, $X_0=Y_0=\sqrt{b(r-1)},Z_0=r-1$:
\begin{align*}
    \dot X&=s(\sqrt{b(r-1)}-\sqrt{b(r-1)})=0\\
    \dot Y&=r\sqrt{b(r-1)}-\sqrt{b(r-1)}-\sqrt{b(r-1)}(r-1)\\
    &=(r-1)\sqrt{b(r-1)}-\sqrt{b(r-1)}(r-1)=0\\
    \dot Z&=\sqrt{b(r-1)}\cdot \sqrt{b(r-1)}-b(r-1)=0 \quad \square
\end{align*}
Finally, we check the third solution, $X_0=Y_0=-\sqrt{b(r-1)},Z_0=r-1$:
\begin{align*}
    \dot X&=s(-\sqrt{b(r-1)}-(-\sqrt{b(r-1)}))=0\\
    \dot Y&=-r\sqrt{b(r-1)}+\sqrt{b(r-1)}-(r-1)(-\sqrt{b(r-1)})\\
    &=(-r+1)\sqrt{b(r-1)}+\sqrt{b(r-1)}(r-1)=0\\
    \dot Z&=\sqrt{b(r-1)}\cdot (-\sqrt{b(r-1)})-b(r-1)=0 \quad \square
\end{align*}
To show that we can linearise our governing equations, we assume that there are small perturbations around our equilibruim solutions, which can be written in the following way: 
\begin{align*}
    \hat{X} = X - X_0 \\
    \hat{Y} = Y - Y_0 \\
    \hat{Z} = Z - Z_0
\end{align*}
We now plug these expressions into the governing equations \ref{goveqX}, \ref{goveqY}, and \ref{goveqZ}:
\begin{align*}
    \dot{X} &= \frac{d}{dt} (X_0 + \hat{X}) = \frac{d}{dt} \hat{X} = sY_0 + s\hat{Y} - sX_0 - s\hat{X} = s\hat{Y} - s\hat{X} \\
    \dot{Y} &= \frac{d}{dt} (Y_0 + \hat{Y}) = \frac{d}{dt} \hat{Y} = rX_0 + r\hat{X} - Y_0 - \hat{Y} - X_0Z_0 - X_0\hat{Z} - \hat{X}Z_0 - \hat{X}\hat{Z} = \hat{X}(r - Z_0)- \hat{Y} - X_0\hat{Z} \\
    \dot{Z} &= \frac{d}{dt} (Z_0 + \hat{Z}) = \frac{d}{dt} \hat{Z} = X_0Y_0 + X_0\hat{Y} + \hat{X}Y_0 + \hat{X}\hat{Y} - bZ_0 - b\hat{Z} = Y_0\hat{X} + X_0\hat{Y} - b\hat{Z}
\end{align*}
where we have dropped terms which are products of two disturbances because they are much smaller than all the other terms, as well as terms which contain purely equilibrium values because these go to 0. \\

These equations are equivalent to the result obtained by carrying out the following matrix vector multiplication:
\begin{equation*}
    \frac{d}{dt}\begin{pmatrix}
    \hat{X} \\ \hat{Y} \\ \hat{Z}
    \end{pmatrix} = \begin{pmatrix}
    -s & s & 0 \\
    r - Z_0 & -1 & -X_0 \\
    Y_0 & X_0 & -b
    \end{pmatrix}\begin{pmatrix}
    \hat{X} \\ \hat{Y} \\ \hat{Z}
    \end{pmatrix} 
\end{equation*}
Knowing that when at least one eigenvalue is bigger than one, the system is unstable, we can see that for the situation where $X_0 = Y_0 = Z_0 = 0$ the system is stable when $r<1$. For the other two situations the $r$ can be much bigger and still have a stable system. The value for $r$ for which we have a stable system is for $r < 25?$. This means that for the last two situations, there are other forces that influence the system more than the density difference due to the temperature gradient.
For $r<1$, there is no velocity field... \\

When we find the eigenvalues of the matrix, we diagonalize the matrix $det(M - \lambda I) = 0$
\begin{align} \label{diagonalize}
    \det \begin{pmatrix}

        d & e - \lambda & f \\
        g & h & i - \lambda
    \end{pmatrix} = (a - \lambda)((e - \lambda)(i - \lambda) - fh) + b(fg - d(i - \lambda)) + c(dh - (e - \lambda)g) = 0    
\end{align}
Looking at expression \eqref{diagonalize} we can see that for us $f = -X_0, g = Y_0, h = X_0$. Because for the two cases of $(X_0, Y_0, Z_0) = (\pm \sqrt{b(r-1)}, \pm \sqrt{b(r-1)}, r-1)$, the only thing that changes is the sign of the $X_0$ and $Y_0$, nothing changes in for the eigenvalues. This is because the term $fh = (-X_0)(X_0)$ stays the same, and $c = 0$, which takes care of the sign switch of $dh = (r - Z_0)X_0 $ and $ (e - \lambda)g = -Y_0 - \lambda Y_0$. This explains why the eigenvalues of these two situations are the same.\\
%
r measures the relative importance of temperature-induced buoyancy forces and frictional forces in our system and therefore shows how much convection will accure. For $r<1$ we have the equilibrium state $X_0 = Y_0 = Z_0 = 0$ (fig. \ref{fig:disturbance0}). When there is at least one possitive eigenvalue, the disturbances will increase and therefore the system is unstable. You can see that for this first equilibrium state, the maximum eigenvalue will stay below $0$ for $r<1$. This means that when $r<1$, no convection will occure, giving a velocity and temperature field as in the situation sketch in the assignment. \\
%
For $r>1$ we have the equilibrium solutions $(X_0, Y_0, Z_0) = (\pm \sqrt{b(r-1)}, \pm \sqrt{b(r-1)}, r-1)$. The maximum eigenvalues of this system stay negative for $r$ being around $r<25$. \\
%
For $r$ becoming much larger, we will see that the eigenvalues are possitive for all the situations. This means that the pertubations will be amplified and therefore we have an instable system. Rayleigh-Bénard convection will occure like in figure \ref{Rayleigh_Bénard_convection}. This will in some places bring warmer water up and colder water down. The line of equal temperature will therefore not be horizontal anymore, but will look more like in figure \ref{Rayleigh_Bénard_convection}.
